文本颜色：

color 属性用于定义文本的颜色。

语法：

div { 
    color: red;
}

开发中最常用的是十六进制

文本对齐：

text-align 属性用于设置元素内文本内容的水平对齐方式。
语法：

div { 
    text-align: center;
}

修饰文本：

语法：

div { 
    text-decoration：underline；
}

重点记住：如何添加下划线 ? 如何删除下划线 ? 其余了解即可.

文本缩进

语法：

div { 
    text-indent：20px；
}
div { 
    text-indent：2em；
}

em 是一个相对单位，就是当前元素（font-size) 1个文字的大小, 如果当前元素没有设置大小，则会按照父元素的 1 个文字大小。

行间距：

line-height 属性用于设置行间的距离（行高）。可以控制文字行与行之间的距离

语法：

p { 
    line-height: 26px;
}
