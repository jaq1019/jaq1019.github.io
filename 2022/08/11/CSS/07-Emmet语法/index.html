简介

Emmet语法的前身是Zen coding，它使用缩写，来提高 html/css 的编写速度，Vscode内部已经集成该语法。

快速生成HTML结构语法

生成标签

  1. 直接输入标签名，按 tab 键即可
    例：输入 div 再按 tab 键，生成
  2. 要生成多个相同标签，加上 * 即可
    例：输入 div*3 再按 tab 键，生成

生成父子级关系的标签，用 >

输入 ul > li 再按 tab 键，生成

生成兄弟关系的标签，用 +

输入 div+p 再按 tab 键，生成

生成带有类名或者id名字的，用 . 和 #

  1. 输入 .demo 再按 tab 键，生成

生成的div 类名有顺序，用自增符号 $

输入 .demo$*3 再按 tab 键，生成

想要在生成的标签内部默认显示出内容用 { } 表示

  1. 输入 div{tymyyds} 再按tab 键，生成
  2. 输入div{$}*3 再按tab 键，生成

快速生成CSS样式语法

CSS 基本采取简写形式即可，例：

  1. 输入 w200 按tab键，生成 width: 200px;
  2. 输入 lh26px 按tab键，生成 line-height: 26px;​

快速格式化代码

Vscode 快速格式化代码: shift+alt+f

保存页面时自动格式化代码设置步骤:

  1. 文件 ——.>【首选项】———->【设置】
  2. 搜索emmet.include
  3. 在“用户”中添加以下语句：

    “editor.formatOnType”: true,
    “editor.formatOnSave”: true
